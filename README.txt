Quacopter simulation 


It is a simple simulation program for simulating quadcopter movements with the help of 
change in rpm of the quadcopter rotors. 

It is made using python and pygame module of python.


setup.py file is used to make the .py file to .exe file for the windows system, it also 
includes some new modules which helps to retrive the font module for displaying the 
text in the pygame screen.
